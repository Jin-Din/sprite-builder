# css sprite

```

	css雪碧图简单制作工具

	可以通过图片，直接生成sprite文件，并且生成代码

	可以通过鼠标点击调整图片位置

	可以添加单张图片，以及删除单张图片

	可以保存为.sprite文件，以后好维护

```

# 修改内容

1. 加入生成 json 配置，以便支持矢量瓦片电子地图中对 sprite 的支持
2. 加入选项框，可以选择输出css、sass、json等内容。
3. 新增：提取sprite雪碧图中的各图标
